List Supported colormaps
~~~~~~~~~~~~~~~~~~~~~~~~~

There are various color maps used by application for various variables. These are carefully selected based on good articles written on choosing color maps in maplotlib and forecast images generated by various global forecasting agencies. Finally, you have to convince your user and hence you should have the option to choose the color map you want. ``wrfplot`` uses ``colormaps`` python module for doing this. This package has got a good level of color maps, including NCAR, GMT MATLAB etc. 

.. tip::
	It is highly recommanded to visit https://pratiman-91.github.io/colormaps/ to look at the colour maps and its name. 

To know the list of supported colormaps, you can use the following command::

	$ wrfplot --list-cmaps

Apply custom colormaps for plots
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

By default, the colormap for specif variable is already set by `wrfplot`. If you do not like the default colormap, then you can choose one of the supported colormaps from the list above. Assuming that you would like to use `temp_diff_18lev` colormap for `T2` (surface Temperature) plot, you can supply `temp_diff_18lev` colormap using ``--cmap`` with following command::
	
	$ wrfplot --cmap temp_diff_18lev --vars 'T2' --input /path/to/wrfout.nc --output /path/to/output_dir
